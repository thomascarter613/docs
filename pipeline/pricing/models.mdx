# Pricing Models

Pricing Models define the offer structures Applied Innovation Corp may use for consulting, implementation, and platform-adjacent services.

The goal is to keep pricing simple enough to sell and structured enough to protect delivery.

---

## Model overview

| Model | Best for | Notes |
| --- | --- | --- |
| Fixed-scope project | Clear deliverables and bounded implementation | Best early default |
| Paid audit | Discovery, workflow mapping, and recommendations | Good entry offer |
| Implementation sprint | One focused workflow or automation | Good for clear pain |
| Monthly retainer | Ongoing optimization and support | Use after trust exists |
| Productized package | Repeatable service with defined outcome | Best long-term service model |
| Hybrid consulting + platform | Implementation plus reusable platform setup | Future model as platform matures |

---

## Paid audit

A paid audit is used when the problem is real but the implementation path is not yet clear.

Includes:

- workflow review
- pain analysis
- automation opportunity map
- risk notes
- recommended implementation path

Best when the prospect needs clarity before committing to implementation.

---

## Fixed-scope project

A fixed-scope project is used when deliverables are clear.

Includes:

- defined workflow
- implementation plan
- build/configuration
- testing
- documentation
- handoff

Best when scope can be bounded.

---

## Implementation sprint

An implementation sprint is a focused engagement around one high-value workflow.

Best when:

- urgency is clear
- scope is narrow
- outcome is specific
- buyer wants quick operational improvement

---

## Monthly retainer

A retainer supports ongoing improvement, support, and optimization.

Best after:

- an initial project succeeds
- trust exists
- the client has recurring needs
- ongoing system maintenance is justified

---

## Productized package

A productized package is a repeatable service with defined scope, timeline, and outcome.

Best when repeated delivery patterns have been proven.

---

## Hybrid consulting + platform

The hybrid model combines service delivery with reusable platform infrastructure.

This should mature gradually as Monad-Workspace and Applied Innovation Corp delivery systems become more proven.

---

## Recommended early pricing approach

Use this sequence:

1. Paid audit
2. Fixed-scope implementation sprint
3. Follow-on support or retainer
4. Productized package
5. Hybrid consulting + platform model

This sequence reduces risk while preserving upside.
