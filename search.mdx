# Search

Use this page as a guide for finding information across the Monad-Workspace docs.

Mintlify provides built-in search through the documentation search bar. Use the search bar or keyboard shortcut to find pages, headings, concepts, templates, decisions, and operating procedures.

---

## How to search

Use search when you need to find:

- a system
- a workflow
- a template
- an ADR
- a governance rule
- a concept
- a pattern
- a lineage record
- a Founder Log process
- an AI Cofounder instruction

Search works best when you use specific terms.

## Search quality rules

To keep search useful:

- Use clear page titles.
- Write specific descriptions.
- Add accurate tags.
- Use consistent terminology.
- Prefer canonical names such as `Founder Log`, `AI Cofounder`, `ADR`, and `Monad-Workspace`.
- Avoid duplicate page titles when possible.
- Use `boost` only for pages that should rank higher.
