# Engagement Templates

Engagement Templates standardize how Applied Innovation Corp structures client work.

They help reduce ambiguity, improve delivery quality, and make engagements easier to scope and repeat.

---

## Core engagement types

| Engagement | Purpose |
| --- | --- |
| AI Workflow Audit | Identify automation and workflow improvement opportunities |
| AI Implementation Sprint | Implement one focused workflow improvement |
| Operating System Buildout | Design and implement a broader internal operating system |
| Documentation and Governance Sprint | Create structure, docs, and decision controls |
| Automation Cleanup | Simplify and stabilize existing automations |
